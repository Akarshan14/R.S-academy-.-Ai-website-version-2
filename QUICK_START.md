# ⚡ Quick Start - Deploy in 5 Minutes

## Step 1: Download/Clone this project
Already done ✅

## Step 2: Test locally (optional)
```bash
pnpm install
pnpm dev
```
Open http://localhost:5173 in your browser

## Step 3: Deploy to Vercel

### Via GitHub (Recommended):
```bash
# Initialize git (if not already)
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit"

# Create repo on GitHub, then:
git remote add origin https://github.com/YOUR_USERNAME/rs-academy.git
git push -u origin main

# Now go to vercel.com and import your GitHub repo
```

### Direct Deploy (No GitHub needed):
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Follow the prompts, done!
```

## Step 4: Done! 🎉

Your website will be live at: `https://your-project.vercel.app`
